Scratch3 Tello ブロックの使い方

DJI Tello を Scratch3.0 から操作するための拡張ブロックの一覧と使い方をまとめました。

準備

  • パソコンと Tello を同じ Wi-Fi（Tello のアクセスポイント）に接続します。
  • Scratch3 Tello 拡張を追加し、ブロックパレットに「Tello」カテゴリが表示されていることを確認します。
接続が不安定な場合は、Tello を再起動し、近距離でテストしてください。
「接続する（につなぐ）」ブロックは 真偽値 ブロックとして動作し、接続が成功すると true を返します。実行直後に条件判定に入れることで、接続できたかを即座にチェックできます。

基本ブロック

  • 接続する
    Tello と接続を開始します。真偽値 ブロックなので、true が返ってきたら接続完了です。
  • 離陸
    Tello が自動で離陸しホバリングします。
  • 着陸
    安全に着陸させます。バッテリー残量が少ない時は早めに実行してください。
  • 前／後／左／右／上／下 に動く (cm)
    指定した距離だけ直線移動します。20〜500 の範囲が目安です。
  • 時計回り／反時計回りに回る (°)
    機体をその場で回転させます。角度は 1〜360° を推奨。
  • 速度を (cm/s) にする
    移動コマンドのスピードを設定します。

ステータス系ブロック

  • バッテリー %
    現在のバッテリー残量を取得します。低い場合は飛行を控えます。
  • 高度 (cm)
    離陸地点からの高さを取得します。
  • 温度 (℃)
    モーター温度を取得します。高温時は休ませてください。
  • 速度 (cm/s)
    現在の移動速度を取得します。

よく使うプログラム例

1. 基本フライト

  • 「接続する」→「離陸」→「前に動く 50cm」→「時計回りに回る 90°」→「着陸」

2. 四角形に飛行する

  • 繰り返し 4 回、前へ 80cm → 右回転 90° を組み合わせて、矩形飛行を実現します。

3. 安全確認ルーチン

  • 飛行前に「バッテリー %」が 30 以上かチェックし、条件を満たさない場合は「着陸」に分岐します。
