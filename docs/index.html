Scratch3 Tello ブロックの使い方

DJI Tello を Scratch3.0 から操作するための拡張ブロックの一覧と使い方をまとめました。

準備と接続の流れ

  • パソコンと Tello を同じ Wi-Fi（Tello のアクセスポイント）に接続します。
  • Scratch3 Tello 拡張を追加し、ブロックパレットに「Tello」カテゴリが表示されていることを確認します。
  • 最初に 接続する ブロックを置き、true が返ってきたことを条件分岐で確認すると安心です。
接続が不安定な場合は、Tello を再起動し、近距離でテストしてください。
「接続する（につなぐ）」ブロックは 真偽値 ブロックとして動作し、成功時は true を返します。接続チェックの条件にそのまま利用できます。

Tello ブロックの見方

基本ブロック

  • 接続する — Tello と通信を開始し、成功すると true を返します。
  • 離陸 / 着陸 — 安全に離着陸を実行します。バッテリーが少ないときは早めの着陸を。
  • 前／後／左／右／上／下 に動く (cm) — 20〜500cm を目安に直線移動します。
  • 時計回り／反時計回りに回る (°) — その場で回転。1〜360° が推奨です。
  • 速度を (cm/s) にする — 移動速度を調整します。狭い室内では低めに設定しましょう。

ステータス系ブロック

  • バッテリー % — 現在のバッテリー残量を取得。30% 未満なら飛行を控えます。
  • 高度 (cm) — 離陸地点からの高さを取得します。
  • 温度 (℃) — モーター温度を確認。高温時は休ませてください。
  • 速度 (cm/s) — 現在の移動速度を取得します。
ブロックパレットは「Tello」カテゴリを開くとフライト操作・ステータス情報がまとまって表示されます。下記の拡大画像も参考にしてください。

よく使うプログラム例と完成イメージ

1. 基本フライト

  • 接続 → 離陸 → 前に動く 50cm → 時計回り 90° → 着陸。
  • 初めて飛ばすときや室内テストに適した最小構成です。

2. 四角形に飛行する

  • 繰り返し 4 回、前へ 80cm → 右回転 90° を組み合わせて矩形飛行を作ります。
  • ループと回転の練習に最適で、コース飛行の基本になります。

3. 安全確認ルーチン

  • 飛行前に バッテリー % が 30 以上か判定し、満たさなければ即座に着陸します。
  • 温度や高度の情報も組み合わせると、より安全なチェックリストを作成できます。

Tello 本体のイメージ

教材や発表スライドに使いやすい機体の写真です。ホバリング写真を追加すると、飛行中の様子も説明しやすくなります。
