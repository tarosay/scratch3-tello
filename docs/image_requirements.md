# Scratch3 Tello 解説ページ向け 画像リスト

HTML を制作する際に掲載したい説明画像の候補です。撮影/キャプチャ時の共通の目安:
- 解像度: 横 1200px 以上（16:9 または 4:3 推奨）
- 形式: PNG または高品質 JPEG
- 文字が読める明るさ・コントラストで撮影

## 1. 編集画面・ブロック周り
1. **Scratch エディター全景 (Tello 拡張追加済み)**
   - 目的: 「Tello」カテゴリがブロックパレットに表示されていることを示す。
   - ポイント: 左のブロックパレットから Tello カテゴリが見えること。右側にステージとスクリプトエリアが写る全体キャプチャ。

2. **Tello ブロック一覧のクローズアップ**  
   - 目的: 主要ブロック名が判読できるようにする。  
   - ポイント: ブロックパレットの Tello セクションをアップで撮影。スクロールが必要なら複数枚に分けても可。

## 2. サンプルスクリプトのキャプチャ
3. **基本フライト例**  
   - 構成: 「接続する → 離陸 → 前に動く 50cm → 時計回りに回る 90° → 着陸」。
   - ポイント: 一連のブロックが縦に並んでいる様子をフルで収める。

4. **四角形飛行ループ**  
   - 構成: 「繰り返し 4 回」の中で「前に動く 80cm」+「時計回りに回る 90°」を組み合わせたもの。  
   - ポイント: 繰り返しブロック内の構造が分かる拡大キャプチャ。

5. **安全確認フロー**  
   - 構成: 「バッテリー %」を 30 以上で判定し、条件を満たさなければ「着陸」する分岐を含む。  
   - ポイント: 条件分岐の形が見えるよう横幅を確保。必要ならスクロール位置を調整。

## 3. ハードウェア写真
6. **Tello 機体の正面/斜め写真**  
   - 目的: 機体の外観紹介。  
   - ポイント: 室内の明るい場所で、プロペラやカメラが見える角度。

7. **ホバリング中の Tello**  
   - 目的: 実際に飛行している雰囲気を伝える。  
   - ポイント: 低高度で静止している瞬間。安全第一で撮影。

8. **Wi-Fi 接続確認画面 (任意)**  
   - 目的: PC が Tello のアクセスポイントに接続していることを示す補足素材。  
   - ポイント: OS の Wi-Fi 設定画面で SSID「TELLO-xxxx」が選択されている様子。

## 4. (必要に応じて) UI 追加素材
9. **拡張機能追加の手順スクリーンショット**  
   - 目的: Scratch で Tello 拡張を追加する操作を説明する際の補足。  
   - ポイント: 拡張追加画面で Tello アイコンをクリックする場面。

10. **エラーメッセージ例 (任意)**
    - 目的: 接続失敗などが発生したときの画面を示し、トラブルシュートに使う。
    - ポイント: 画面が見やすい解像度で、原因が読めること。

上記のうち必須度が高いのは 1〜5 と 6 です。7〜10 は補足やトラブルシュート用にあると便利です。
